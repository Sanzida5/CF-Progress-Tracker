## About the Project

I created this project to make my competitive programming practice more organized. Instead of keeping solved problems and goals in a notebook or spreadsheet, this dashboard helps me record problems, check my progress and understand which topics need more practice.

The app is focused on Codeforces practice, but the same idea can also be used for other programming platforms.

## Features

- Dashboard for practice overview
- Add and manage solved problems
- Filter problems by rating, topic, and status
- Track current rating and target rating
- Daily practice target and streak tracking
- Goal management section
- Analytics page with progress charts
- Weak topic tracking
- Profile section for user information
- Light and dark mode
- LocalStorage support, so data stays saved in the browser
- Responsive layout for different screen sizes

## Tech Stack

- React.js

## How to Run Locally

Clone the repository:

```bash
git clone https://github.com/Sanzida5/CF-Progress-Tracker.git
```

Go to the project folder:

```bash
cd CF-Progress-Tracker
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
